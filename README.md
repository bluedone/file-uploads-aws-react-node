## Installation Instructions :point_right:

1. Clone the repo using `git clone https://github.com/imranhsayed/file-uploads-aws-react-node`
2. `cd file-uploads-aws-react-node`
3. `npm install`
4. `cd client`
5. `npm install`
7. `cd ..`
8. `npm run dev`
